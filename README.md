A clean, structured React project showcasing reusable components, state management, and modern UI design. This Todo App allows users to create, update, delete, and manage tasks with a smooth user experience and scalable folder architecture.

Features
. Add new tasks 
. Mark tasks as completed
. Delete tasks
. Responsive layout
. Clean, modular file structure
. Uses React hooks (useState, useEffect)
. Fully reusable components

What This Project Demonstrates
. Clean React folder architecture
. Separation of concerns.
. Controlled forms
. State lifting
. Basic UI/UX design

Future Improvements
. Local storage sync
. Filters (Completed/Pending/All)
. Animations
. Drag-and-drop reordering

Technologies Used
. React + Vite
. JavaScript (ES6+)
. CSS
.Git & GitHub

How to Run the Project
npm install
npm run dev
